# Real-Time Face Recognition System

A real-time face recognition system using webcam, built with Python, OpenCV, and face_recognition library.

## Features
- Real-time face detection and recognition
- Save/load face encodings for faster startup
- Command-line configuration
- FPS optimized processing
- Screenshot capture
